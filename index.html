A formal abstract summarizing the project's purpose, methodology, and findings will be published here. Keep Attendance is a student-developed AI-powered attendance system built to address real-world inefficiencies in manual tracking through computer vision and machine learning.

The Solution

Keep Attendance uses AI and computer vision to see who's present — automatically, accurately, and in real time.

✓
AI-powered visual recognition — Analyzes live video to detect presence with high accuracy.
✓
Automatic detection & logging — No manual input. Presence is confirmed and timestamped in real time.
✓
Simple web dashboard — View attendance, history, and live status from any browser on your network.
✓
Portable & easy to set up — Low-cost hardware, minimal configuration. Deploy anywhere in minutes.
